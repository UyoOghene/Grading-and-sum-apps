Sum App Assignment

QUESTION 3 - Summation of inputted numbers Create an app, which continiously takes several numbers as input using a prompt, till you input zero or cancel and then it returns the sum as output. e.g. 1. input = 1, 2, 3, 5 output = 1 + 2 + 3 + 5 = 11 e.g. 2. input = 4, 6, 20, 3 output = 4 + 6 + 20 + 3 = 33 e.g. 3. input = 6, 5, 10, 20, 7 output = 6 + 5 + 10 + 20 + 7 + 1 = 49
